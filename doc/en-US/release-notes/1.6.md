# Chats 1.6 Release Notes

> Release date: 2024-06-30 (approximately 60 commits since 1.5.4.844)

Version 1.6 is a significant update featuring .NET 9 upgrade, enhanced reasoning model support, improved image upload capabilities, better OpenAI compatibility, and numerous user experience optimizations and bug fixes.

## Highlights

- Upgraded to .NET 9: Full upgrade to the latest .NET 9 framework for improved performance and security
- Enhanced Reasoning Models: Added support and optimizations for o3-pro, o4-mini, Gemini Think, and more
- Image Feature Improvements: History image upload, phone camera capture, image editing model selection, forced 3:2/2:3 ratios
- New Model Support: GPT-5 series, Qwen3 series, GLM-4.5 series, new Kimi models, Grok-4, Codex-mini
- OpenAI Compatibility: Fixed response API JSON Schema issues, improved status codes, EndUserId fixes
- Azure Deployment: New one-click deployment script for simplified Azure environment setup
- User Experience: Fullscreen hotkey, optimized message selection, UI detail improvements
- Deployment Optimization: Migration to R2 storage, Ubuntu 22.04, build timeout adjustments

## Technical Improvements

### Framework Upgrade
- **Full .NET 9 Upgrade**: Leveraging the latest .NET 9 features for improved performance, security, and developer experience

### Reasoning Model Support
- **o3-pro Series Optimization**: Enhanced reasoning summary and async API support
- **o3/o4-mini**: Fixed image input support
- **Gemini Think**: Support for Google's Gemini Think reasoning with optimized effort level mapping

### OpenAI Compatibility
- **JSON Schema**: Fixed JSON Schema compatibility in response API
- **Status Code Returns**: Proper status code returns in OpenAI compatible SDK
- **GPT-5 Series**: Comprehensive native support for GPT-5 and GPT-5-chat
- **EndUserId**: Fixed EndUserId issues in gpt-image-1

### Image Processing
- **Ratio Control**: Support for forced 3:2 and 2:3 image ratios
- **Model Selection**: Choose specific models when editing images
- **Camera Capture**: Mobile support for direct camera capture upload
- **History Upload**: Upload images within chat history

### Deployment & Infrastructure
- **Storage Migration**: Migrated from MinIO to Cloudflare R2 for improved storage performance
- **Azure One-Click Deployment**: New automated deployment script for simplified Azure configuration
- **Ubuntu 22.04**: Switched to more stable Ubuntu 22.04 LTS
- **Build Optimization**: Increased build timeout to 2 hours for successful large builds

### SDK Consolidation
- Removed Azure OpenAI .NET SDK, unified to OpenAI SDK
- Simplified dependencies, reduced maintenance costs

## Deployment Notes

### Azure One-Click Deployment

Version 1.6.5 introduces an Azure one-click deployment script for quick deployment in Azure environments. For details, please refer to:
- [Azure Bicep Deployment Documentation](../azure-bicep.md)

### Storage Configuration

Starting from version 1.6.7, Cloudflare R2 is recommended for object storage. Please refer to relevant documentation for storage configuration.

### .NET 9 Requirements

Version 1.6.1 and later require .NET 9 runtime. Please ensure your deployment environment has .NET 9 installed.

---

## Upgrade Recommendations

1. **Backup Data**: Always backup database and configuration files before upgrading
2. **Check Dependencies**: Ensure .NET 9 runtime is installed in the environment
3. **Storage Configuration**: If using object storage, consider migrating to Cloudflare R2
4. **Testing**: Perform comprehensive functional testing after upgrade

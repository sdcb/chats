# Chats 1.1.0 Release Notes

> English version coming soon. Please refer to the [Chinese version](../../zh-CN/release-notes/1.1.md) for now.

## Quick Summary

Version 1.1.0 introduces support for Azure OpenAI reasoning models (o3/o4-mini) through the new Response API, along with SDK upgrades and various optimizations.

### Key Highlights

- **Azure OpenAI Reasoning Models**: New Response API integration for o3/o4-mini with reasoning summary support
- **SDK Upgrades**: OpenAI .NET SDK and Azure OpenAI SDK upgraded to 2.2.0-beta.4
- **Reasoning Content Optimization**: Improved formatting with double line breaks between thinking blocks
- **Default Prompts Optimization**: Enhanced system default prompts for better conversation quality
- **Image Upload Optimization**: Synchronized upload to Minio for faster access in China

For detailed information, please refer to the [Chinese version](../../zh-CN/release-notes/1.1.md).

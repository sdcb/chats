# Chats 版本发布历史

本页面索引了 Chats 项目的所有主要版本发布说明，从最新版本到早期版本。每个版本都包含了该版本的核心功能、改进和修复。

---

## [1.7.2](1.7.2.md) - 2025-10-27 ⭐ 最新版本

**核心亮点**：图像生成流式支持 · 代码执行功能 · API 测试框架

- 🖼️ **图像生成流式支持**：`gpt-image-1` 和 `gpt-image-1-mini` 支持实时查看生成进度
- 🔧 **代码执行优化**：Gemini 代码执行结果以工具调用形式清晰展示
- 🧪 **API 测试框架**：新增 `Chats.BE.ApiTest` 项目，全面测试 OpenAI 兼容 API
- 🤖 **新模型**：添加 `gpt-5-codex`、`gpt-5-pro`、`gpt-image-1-mini` 模型
- ⚙️ **配置增强**：ChatConfig 新增 `CodeExecutionEnabled` 字段
- 📦 **依赖升级**：升级 11 个第三方依赖包

[查看完整发布说明 →](1.7.2.md)

## [1.7.1](1.7.1.md) - 2025-10-13

**核心亮点**：安全审计日志 · 发送体验优化 · 工具调用展示

- 🔒 **安全审计**：完整的登录尝试记录与速率限制监控
- 📤 **发送体验**：支持多种发送模式（发送、继续、重新生成）与移动端适配
- 🔨 **工具调用**：独立的 `ToolCallBlock` 组件，更清晰的参数与结果展示
- ⚡ **性能优化**：生成信息按需加载，减少初始加载时间
- 📱 **移动端**：更好的响应式布局与触控体验
- 👨‍💼 **管理后台**：消息内容查询、用户用量统计优化

[查看完整发布说明 →](1.7.1.md)

## [1.7.0](1.7.md) - 2025-09-20 🎉 重大更新

**核心亮点**：MCP 全面支持 · 数据库重构 · 拖拽排序

- 🔌 **MCP 支持**：服务端/前端联动、用户级授权、工具抓取、会话调用与权限校验
- 🛠️ **工具调用强化**：SSE 事件更丰富、消息内容新增工具请求/响应类型
- 🗄️ **数据库重构**：Message→ChatTurn/Step 分层、历史数据迁移（破坏性变更）
- 🎨 **拖拽排序**：模型/密钥/预设支持拖拽排序
- 📊 **Mermaid 支持**：新增 Markdown Mermaid 渲染，支持暗/亮主题
- 🖼️ **图片尺寸**：新增常用尺寸选择（1024×1024、1536×1024、1024×1536）
- 🔄 **重新生成**：支持单条或整段消息重新生成

[查看完整发布说明 →](1.7.md)

## 版本命名说明

从 1.0.0 开始，Chats 采用语义化版本号（Semantic Versioning）：

- **主版本号（Major）**：重大架构变更或破坏性更新（如 1.0.0 → 2.0.0）
- **次版本号（Minor）**：新增功能，向下兼容（如 1.0.0 → 1.1.0）
- **修订号（Patch）**：问题修复和小幅优化（如 1.0.0 → 1.0.1）
